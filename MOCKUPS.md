# Mockups oficiales

Este documento registra la relación entre los mockups visuales aprobados y las páginas funcionales de Web-Labs Inducción Operacional.

Los mockups son referencia visual y de QA. No deben usarse como imágenes estáticas pegadas dentro de la web productiva.

## Uso permitido

Los mockups deben utilizarse como referencia para:

- jerarquía visual;
- layout;
- proporciones;
- navegación;
- tratamiento de tarjetas;
- estados visuales;
- modo Boldo;
- modo Ámbar;
- composición responsive;
- QA visual.

## Uso no permitido

No deben utilizarse como imágenes fijas dentro de la interfaz productiva.

Cada mockup debe traducirse progresivamente a componentes React reutilizables.

## Criterio legal-operacional

La plataforma es formativa. No debe registrar ni exponer datos reales de víctimas, PSC, domicilios, teléfonos, coordenadas, folios, causas ni antecedentes identificables.
